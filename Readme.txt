ABOUT
This project code named Kairos [http://en.wikipedia.org/wiki/Kairos] is a single page application that acts as a repository for all opportunities

TECHNOLOGIES
-----------------------------------------------------------------------------------------------
You will need Visual Studio 2013 and dependencies below to be able to work and run this project
-----------------------------------------------------------------------------------------------
Server Side : ASP.net Web API 2.0,Entity Framework 6.0.2,SQL Server (LocalDb)
Client Side : HTML 5, Angular.js
Css			: Bootstrp with Glyphicons - Bootstrap theme - Cerulean from www.bootswatch.com
IoC			: Unity WebApi 5.1
Testing		: Behaviour Driven Development using SpecFlow
			  MSTest - Unit testing framework
			  Selenium with Chrome Driver - For UI Automation
		 
SOURCE CONTROL
GitHub : https://github.com/ankushgit/Kairos.git
